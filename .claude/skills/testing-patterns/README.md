# Testing Patterns

## Skill Summary

This skill provides a repeatable way to verify PHP business features, workflow changes, and access-control behavior.

## Usage Scope

Use it for manual validation, automated test planning, regression checks, and completion evidence.

## Example Invocation Scenarios

- Verifying a new department task board
- Testing who can close or reassign a task
- Confirming task comments persist correctly
- Checking that unauthorized users cannot open restricted pages

## Works Well With

- `test-automator`
- `php-mvc-engineer`
- `security-engineer`
- `workflow-domain-analyst`

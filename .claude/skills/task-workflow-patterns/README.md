# Task Workflow Patterns

## Skill Summary

This skill structures how the portal models and implements task lifecycle behavior across departments.

## Usage Scope

Use it for task states, transitions, assignments, approvals, escalations, and audit-aware workflow behavior.

## Example Invocation Scenarios

- Designing a manager approval step
- Adding reassignment rules between departments
- Preventing invalid task closure
- Building status-based queue pages for each department

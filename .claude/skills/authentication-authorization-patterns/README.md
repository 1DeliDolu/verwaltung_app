# Authentication and Authorization Patterns

## Skill Summary

This skill covers secure identity, session, and access-control behavior for the internal department portal.

## Usage Scope

Use it for login, logout, session management, permission checks, department access, and protected workflow actions.

## Example Invocation Scenarios

- Restricting finance pages to finance members
- Allowing managers to review cross-department tasks
- Adding a secure task approval action
- Reviewing logout and session timeout handling

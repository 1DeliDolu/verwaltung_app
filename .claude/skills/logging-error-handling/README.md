# Logging and Error Handling

## Skill Summary

This skill defines how to surface errors safely and how to keep logs useful for operational support and debugging.

## Usage Scope

Use it for exceptions, validation failures, user feedback, and operational diagnostics.

## Example Invocation Scenarios

- Handling task submission validation failures
- Logging failed cross-department reassignment attempts
- Reviewing how unexpected service errors are surfaced
- Improving support visibility for production incidents

# PHP MVC Patterns

## Skill Summary

This skill defines how to structure request handling, business logic, and rendering for a maintainable PHP MVC-style application.

## Usage Scope

Use it for controllers, services, views, form handling, and request-to-response design.

## Example Invocation Scenarios

- Building a department dashboard page
- Adding a task assignment form
- Refactoring a large controller method into a service
- Standardizing how validation errors are returned to views

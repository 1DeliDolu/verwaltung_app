# 54. Mail Header And Theme

1. `resources/views/layouts/app.php` mail sayfasinda da ortak ust header gosterilecek sekilde guncellendi.
2. Mail ekraninin ozel layout davranisi korunurken footer gizli kaldi ve workspace genisligi ayni tutuldu.
3. `resources/views/mail/index.php` renkleri siyah temadan ana uygulamadaki bej, krem ve kiremit tonlarina cekildi.

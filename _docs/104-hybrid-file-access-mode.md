## 104. Hybrid File Access Mode

1. File server kullanimi hibrit mod olarak netlestirildi.
2. Ayni departman dosyalari hem uygulama icindeki web-dateibrowser uzerinden hem de Samba/SMB uzerinden erisilebilir olacak sekilde ifade edildi.
3. `Infrastruktur` sayfasindaki file server kartina hibrit erisim aciklamasi eklendi.
4. Web-dateibrowser sayfasina `Ein Share, zwei Wege` bolumu eklendi.
5. Bu bolumde web erisimin ne zaman uygun oldugu ve SMB erisimin ne zaman avantajli oldugu aciklandi.
6. Demo portu `localhost:1445` icin bunun HTTP degil Samba/SMB servisi oldugu acik bicimde belirtildi.

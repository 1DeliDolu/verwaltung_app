# 78. Calendar Browser Reminder

1. `resources/views/pages/calendar.php` icindeki termin kartlarina event meta verileri eklendi.
2. Sayfaya, termin baslamadan 1 saat once tarayici uzerinden hatirlatma gosteren JavaScript akisi eklendi.
3. Hatirlatma durumu `localStorage` ile tutuldu; ayni termin icin tekrar bildirim engellendi.

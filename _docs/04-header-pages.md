# 04 Header Pages

## Yapilan Islem

Header navigasyonunun baglanacagi public sayfalar eklendi.

## Eklenen veya Guncellenen Parcalar

- `app/Controllers/PageController.php`
- `routes/web.php`
- `resources/views/pages/news.php`
- `resources/views/pages/calendar.php`

## Sonuc

- `News` icin public sayfa olustu.
- `Calendar` icin public sayfa olustu.
- Ana rota misafir kullanici icin `News` sayfasina baglandi.

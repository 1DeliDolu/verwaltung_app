## 82. Calendar Edit Visibility

1. Event kartlarindaki aksiyon metni `Bearbeiten` yerine `Termin aktualisieren` olarak netlestirildi.
2. Edit linki `#calendarCreateForm` anchor'i ile form alanina baglandi.
3. Takvim edit modunda ust buton metni de `Termin aktualisieren` olarak degisiyor.
4. Edit sayfasi acildiginda form bolumune otomatik scroll davranisi eklendi.

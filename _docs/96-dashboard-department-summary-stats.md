## 96. Dashboard Department Summary Stats

1. Dashboard abteilungs kartlarina sayisal ozet alanlari eklendi.
2. Tum bolumlerde ortak olarak departman dokuman sayisi ve dosya sayisi gosterilmeye baslandi.
3. IT bolumu icin ek olarak `Verwaltete Konten` metrigi eklendi; bu alan IT tarafindan olusturulmus hesap sayisini gosterir.
4. HR bolumu icin ek olarak `Mitarbeiter` ve `Personalakten` metrikleri eklendi.
5. Gerekli sayilar acik model metodlari ile toplandi; dashboard icinde ham sorgu veya view icinde hesaplama yapilmadi.
6. Dashboard kartlari icin yeni ozet tile gorunumu eklendi, boylece tum bolumlerde ayni bilgi mimarisi korundu.

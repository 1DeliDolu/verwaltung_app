# 74. Calendar UI And Flow

1. `app/Controllers/CalendarController.php` ile `/calendar` sayfasi DB verisi ve form akisina tasindi.
2. `resources/views/pages/calendar.php` statik kartlardan cikartilip event formu ve tarih sirali liste gorunumu ile yeniden kuruldu.
3. Kullanicilar takvimden yeni termin girebilir, departman secebilir ve eventler yakin tarihten uzak tarihe siralanmis sekilde gorur.

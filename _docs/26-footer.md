# 26 Footer

## Yapilan Islem

Uygulama layout icine ortak footer eklendi.

## Eklenen veya Guncellenen Parcalar

- `resources/views/partials/footer.php`
- `resources/views/layouts/app.php`

## Sonuc

- Tüm sayfalarda ortak footer gorunuyor.
- Hızlı linkler ve ortam bilgisi footer icine eklendi.
- Demo modda footer icinde `Probe / Demo Umgebung` bilgisi gorunuyor.

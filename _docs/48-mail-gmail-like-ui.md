# 48. Mail Gmail Like UI

1. `resources/views/mail/index.php` Gmail benzeri compose + directory + inbox/sent duzeni ile yeniden kuruldu.
2. `app/Controllers/InternalMailController.php` coklu alici secimini ve attachment upload alanini isleyen akisa guncellendi.
3. Inbox ve sent panelleri kullanicinin kendi mesajlarini gosterecek sekilde korunurken attachment isimleri ve coklu alici listesi arayuzde gorunur yapildi.

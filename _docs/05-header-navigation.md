# 05 Header Navigation

## Yapilan Islem

Uygulama layout icine ortak header navigasyonu eklendi.

## Eklenen veya Guncellenen Parcalar

- `resources/views/partials/header.php`
- `resources/views/layouts/app.php`

## Sonuc

- Header icinde uygulama adi gorunur hale geldi.
- Misafir durumda `News`, `Calendar`, `Login` linkleri gosteriliyor.
- Giris yapildiginda `Login` yerine `Dashboard` linki gosteriliyor.
- Ust navigasyon tum sayfalarda ortak hale getirildi.
